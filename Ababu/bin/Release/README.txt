Installation.

Before installing this software please read this few instruction.
1) If this is your first installation, be sure to setup a MySQL / MariaDB database. 
You can find the script to create an empty database in the installation folder.
Please Visit http://www.oldauntie.org/ababu-installation/

2) If you have already installed Ababu don't forget to check the connection setting.
You can do this from Login Form > Database Settings (Gear icon)
